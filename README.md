## YOUTUBE CLONE

A responsive YouTube homepage clone built using HTML and CSS. This project replicates the UI of YouTube, focusing on layout, styling, and responsiveness.
---

## Features

-  Fully responsive design
-  YouTube-like video grid layout
-  Header with search bar and icons
-  Sidebar navigation menu
-  Clean and modern UI design

---

## Tech Stack

- HTML5
- CSS3 (Flexbox & Grid)
